# Andrés Guerrero 106

Sitio estático con HTML + CSS + JS para la campaña. Incluye páginas públicas, estilos modulares y conexión opcional a Firebase (CDN). Usa assets/css/main.css para cargar todos los estilos y assets/js/script-global.js para las interacciones.

## Ejecutar

Abre index.html en el navegador. No requiere build.

## Estructura

- assets/css: estilos base y por página
- assets/js: scripts globales y config Firebase
- assets/img: imágenes de la campaña
- pdf: documentos descargables

## Notas

- Los PDFs e imágenes actuales son marcadores de posición.
- Firebase es opcional: si falla, usa mocks y no rompe la UI.
