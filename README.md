# Andrés Guerrero 106

Sitio estático de campaña (HTML + CSS + JS) para la lista 106. Incluye páginas públicas, estilos modulares, iconografía temática, descargas de PDFs y conexión opcional a Firebase (CDN) con fallback a mocks para que el formulario funcione siempre.

## Ejecutar

Abre index.html en el navegador. No requiere build.

## Estructura

- assets/css: estilos base y por página
- assets/js: scripts globales y config Firebase
- assets/img: imágenes de la campaña
- pdf: documentos descargables

## Notas

- PDFs e imágenes son marcadores de posición.
- Firebase es opcional: si falla, usa mocks y no rompe la UI.
- Paleta y espaciados están alineados en todas las páginas; botones y anclas listos para móvil y desktop.
